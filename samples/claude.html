전체 복사는 확실히 너무 사이즈가 크다. "질문 1개 + 답변 1개"만 뽑을 수 있도록 알려줘.

질문-답변 1쌍만 추출하는 방법

ChatGPT

javascript
// 개발자 도구 콘솔에서 실행
// 마지막 Q&A 쌍 추출
const articles = document.querySelectorAll('article[data-testid^="conversation-turn"]');
const lastTwo = Array.from(articles).slice(-2);
copy(lastTwo.map(el => el.outerHTML).join('\n\n<!-- separator -->\n\n'));

특정 턴을 원하면:

javascript
// 첫 번째 Q&A (index 0, 1)
const articles = document.querySelectorAll('article[data-testid^="conversation-turn"]');
copy([articles[0].outerHTML, articles[1].outerHTML].join('\n\n'));

범용 탐색 스크립트

selector가 안 맞으면 이걸로 구조 파악:

javascript
// 현재 페이지에서 "user", "assistant", "human", "model" 등 
// role 관련 속성을 가진 요소 찾기
const allElements = document.querySelectorAll('*');
const roleElements = Array.from(allElements).filter(el => {
  const attrs = Array.from(el.attributes);
  return attrs.some(attr =>
    /role|author|message|turn/i.test(attr.name) ||
    /role|author|message|turn|user|assistant|human|model/i.test(attr.value)
  );
});

console.log('Found elements:', roleElements.length);
roleElements.slice(0, 10).forEach(el => {
  console.log(el.tagName, Object.fromEntries(
    Array.from(el.attributes).map(a => [a.name, a.value])
  ));
});
