What is TypeScript?

TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale 1.

Key Features

  • Static typing: Adds type annotations to JavaScript
  • Compile-time checks: Catches errors before runtime
  • IDE support: Better autocomplete and refactoring 2

TypeScript code compiles to plain JavaScript, which runs anywhere JavaScript runs.

Show me a code example

Here's a simple TypeScript example:

interface User {
  name: string;
  age: number;
}

function greet(user: User): string {
  return `Hello, ${user.name}!`;
}

const user: User = { name: "Alice", age: 30 };
console.log(greet(user));

This demonstrates interfaces and type annotations 3.
