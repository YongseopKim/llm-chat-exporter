Hello, can you explain what a blockchain is?

A blockchain is a distributed ledger technology that maintains a continuously growing list of records called blocks.

  • Each block contains a cryptographic hash of the previous block
  • Transaction data is stored in a tamper-resistant way
  • No single entity controls the entire chain

This makes blockchains ideal for recording transactions in a verifiable and permanent way.

Can you show me a simple Python example?

Here's a simple Python example of a basic block structure:

import hashlib
import time

class Block:
    def __init__(self, data, previous_hash):
        self.timestamp = time.time()
        self.data = data
        self.previous_hash = previous_hash
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_string = f"{self.timestamp}{self.data}{self.previous_hash}"
        return hashlib.sha256(block_string.encode()).hexdigest()

This creates a block with timestamp, data, and cryptographic hash linking.
